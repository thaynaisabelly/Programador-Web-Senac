Propriedades de Posição

Sou Sticky. Alterno entre relativee fixed, dependendo da posição de rolagem. Sou posicionado em relação até que uma determinada posição de deslocamento seja encontrada na viewport - então ele "gruda" no lugar (como position:fixed).

Sou Static. Elementos static não são afetados pelas propriedades superior, inferior, esquerda e direita.

Sou Relative. Definir as propriedades superior, direita, inferior e esquerda de um elemento relative fará com que ele seja ajustado para fora de sua posição normal.

Sou Absolute. Posicionado em relação ao ancestral posicionado mais próximo. Posso sobrepor outros elementos.

Sou Fixed. Posicionado em relação à janela de visualização, o que significa que sempre permaneço no mesmo lugar, mesmo que a página seja rolada. As propriedades top, right, bottom e left são usadas para posicionar o elemento.

Propriedade Descrição
Top Afeta a posição vertical de um elemento posicionado
Left Afeta a posição horizontal de um elememto posicionado
Right Afeta a posição horizontal de um elemento posicionado
Bottom Afeta a posição vertical de um elemento posicionado
Float Especifica como um objeto deve flutuar
Width Especifica largura
Height Especifica a altura
Clip Especifica um retângulo para recortar um elememto posicionado
Position Especifica o tipo de posicionamento usado
Clear Especifica o que deve acontecer com o elemento flutuante
